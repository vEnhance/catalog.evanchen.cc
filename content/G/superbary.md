This unit is essentially a harder version of the bary unit. It includes some challenging problems that can take an extremely long time to bash if not set up correctly. 

It is recommended that one completes one (if not both) of the D-level bary units before attempting this one.

## Philosophy
Only bash when you need to...

The unit introduces how to optimize bashing with synthetic observations, projective geometry, etc. Remember to not just plow through a bash without actually thinking about the problem.

## Notable problems
Definitely very tedious to bash these, but a good setup can make things a bit better.

* [TSTST 2017/5](https://aops.com/community/p8526136), a walkthrough which has both an interesting bary solution and an instructive synthetic solution;
* [USA TST 2004/4](https://aops.com/community/p456576), an interesting application of DDIT;
* [USA TST 2015/6](https://aops.com/community/p4628087), a standard MMP example-- high degree and choice of special requires a bit of thought.
* [Russia 2010](https://aops.com/community/p2014656), a cool problem that uses 3D bary along with Ptolemy's in 3-dimensional space.
* [USAMO 2016/3](https://aops.com/community/p6213572), an Evan amerigeo that's "rumored to be bashable";
